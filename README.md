# 기능

### (1) 화면구성
- 입력값 및 결과값 출력 화면
- 숫자 3개마다 ,(쉼표) 붙이기 (ex: 1,000,000)
### (2) 버튼구성
 - 숫자 0~9
 - ‘+,-,*,/,%’ 연산 및  ‘=’ 추가
 - 연산 버튼 추가시 숫자 입력 화면의 숫자가 사라지고,
   결과출력 화면에 숫자와 연산자가 함께 출력됨
 - 초기화, 한글자 씩 삭제, 계산 결과 확인,
    -기호 붙이기/빼기, 내역 보기
 - 내역 보기 버튼 클릭시 메시지 박스로 지난 계산식,
   결과출력(최대 5개까지 보임)
